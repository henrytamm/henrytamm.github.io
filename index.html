Web SDK Time Tracking

Full API reference for time tracking in the Data 360 Web SDK — configuration options, implementation examples, event structure, and lifecycle management.

# Time Tracking

Time tracking captures user engagement metrics by
monitoring active time spent on pages and catalog
items. Events are sent when configurable time
thresholds are crossed.

## Configuration Options

| Option Name         | Type    | Default   |
| ------------------- | ------- | --------- |
| `enabled`           | boolean | `false`   |
| `thresholds`        | array   | See below |
| `activityTimeout..` | number  | `5000`    |

## Implementation Examples

```javascript
SalesforceInteractions.init({
  dataCloud: {
    timeTracking: {
      enabled: true,
      thresholds: [
        { label: 'ENGAGED', threshold: 15000 },
        { label: 'HIGH_ENGAGEMENT', threshold: 60000 }
      ]
    }
  }
});
```

Connector Guide

Confluence Unstructured Connector

End-to-end guide for ingesting unstructured data from Atlassian Confluence Cloud into Data 360 — setup, limitations, troubleshooting.

# Confluence Unstructured Connector (Beta)

Use the Confluence Unstructured connector to ingest
unstructured data from Atlassian Confluence Cloud
into Data 360.

## Supported Content Types

1. Pages (including nested/child pages)
2. Blog posts
3. Comments (on pages and blog posts)
4. Attachments

## Known Limitations

**Confluence Cloud Only**
This connector supports Atlassian Confluence Cloud
(SaaS) only. Confluence Data Center and Server
are not supported.

**Single Space per Data Lake Object**
Each UDLO ingests content from a single space.
To ingest from multiple spaces, create separate
UDLOs for each space.

Mobile SDK Guide

Capture Engagement Data (Mobile)

Implementation guide for capturing engagement events via the Engagement Mobile SDK — structured/unstructured events, iOS and Android code samples.

# Capture Engagement Data

The Data 360 Module for the Engagement Mobile SDK
enables the collection of rich engagement data
from customer interactions within your mobile app.

## Supported Event Types

### Structured Events
Pre-configured with Mobile Connector schema:
- Cart Event: Track shopping cart interactions
- Catalog Event: Monitor product engagement
- Order Event: Capture purchase data

### Unstructured Events
No predefined schema. Create custom event
definitions for Data 360 ingestion.

```swift
// iOS - Structured AddToCartEvent
SFMCSdk.track(AddToCartEvent(
    lineItem: LineItem(
        catalogObjectType: "Product",
        catalogObjectId: "product-1",
        quantity: 1,
        price: 20.0
    )
))
```
